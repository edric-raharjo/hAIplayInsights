Understanding Gini Impurity

Gini Impurity measures how mixed classes are in a dataset node:

  • Gini = 0: Perfect purity (all samples are one class)
  • Gini = 0.5: Maximum impurity (50/50 split for 2 classes)
  • Formula: Gini = 1 − Σ(pi)2, where pi = proportion of class i

Information Gain = Parent Gini − Weighted Child Gini. A good split has high information gain (low child Gini).
